Is the node alive?

Published: Fr 27 Oktober 2017
By riccomatx

In freifunk.

tags: freifunk

I'd like to check if the node set up last week is still alive, and if so, how much activity it had.

Berlin Freifunk network

There is a website where I get a map and if I go to Tempelhofer Feld, in Berlin, I can see on the right side of the park there are 3 nodes: thf-ap[123]. Those are the sector antennas. And I can check their details...

Map of nodes in Tempelhofer Feld area

Indeed, if I go to the monitor for thf-ap1 I can see several graphs. One of them is the Internet Traffic. There is one graph per network interface (duplicated: one for bytes and other for packets), and I don't know how the interfaces are organised.

Details of one of the nodes in Tempelhofer Feld.

Node status

BUT I can see that the reporting is almost continuous, which means that there is no downtime at night when the wind turbine stops.

There is another link to the wiki page for the Freifunk node. I love it, because I can find out a little bit more info on how it was funded, and how is the network configuration between all the four antennas.

And there is a third link to details on the node on openwifimap. There I can see the transmission power is 8dBm.

The bad news are that there is no uplink with the other node in the airport building. Maybe the parabolic dish is not pointing properly.
